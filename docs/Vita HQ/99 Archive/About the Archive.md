# About the Archive

**What is this?** The retirement home for superseded documentation. Empty today — by design, since Vita HQ was created 2026-07-06 and nothing has been superseded yet.

**Why does it exist?** The vault's rule is **never destroy useful documentation**. When a page is replaced or a direction is abandoned, the old version moves here instead of being deleted — history stays queryable.

---

## Rules

1. **Archive, don't delete.** Move the superseded file here; prefix it with its retirement date: `2026-07-06 Old Page Name.md`.
2. **Leave a tombstone note** at the top of the archived file: what replaced it and why (link the [[Decision Log]] entry if one exists).
3. **Archived pages are never updated** — they are frozen history.
4. Superseded *decisions* don't come here — they stay in the [[Decision Log]], superseded by newer rows.
