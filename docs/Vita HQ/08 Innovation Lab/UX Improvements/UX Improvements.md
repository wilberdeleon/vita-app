# UX Improvements

**Category of:** [[Innovation Lab]]

**Scope:** Interaction, flow, and usability refinements to product areas that already exist — friction reduction, navigation tweaks, micro-interactions. Not new capabilities (those belong in a domain category); this is about making existing capabilities easier or more pleasant to use. Judge every idea here against [[UX Principles]].

**Examples of what fits:** faster food logging shortcuts, smarter default quick-actions on [[Dashboard]], a better empty state, gesture shortcuts in [[My Journey]].
