# Premium Features

**Category of:** [[Innovation Lab]]

**Scope:** Ideas built specifically to justify a paid tier — the feature *is* the monetization argument. Distinct from [[Monetization]], which covers pricing mechanics rather than feature content.

**Examples of what fits:** an advanced analytics view gated to Pro, priority Atlas response depth, exclusive Journey themes, early access to new modules.

**Standing caution:** per [[Core Principles]] #13, health-safety-relevant functionality should never sit behind a paywall — premium ideas should add depth and delight, not gate wellbeing.
