# Health Features

**Category of:** [[Innovation Lab]]

**Scope:** Health and longevity metrics and tracking beyond today's core loop (weight, nutrition, water). This is where Vita's "personal operating system for health" ambition ([[Long-Term Vision]]) lives as concrete proposals.

**Examples of what fits:** Health Age, Biomarker Age, Longevity Dashboard, sleep tracking, recovery scoring, any new metric that measures the body rather than logs a behavior.

**Standing caution:** health claims carry real credibility and trust stakes ([[Core Principles]] #5). Every idea here should be honest about what's scientifically defensible versus aspirational — see each note's Dependencies section.
