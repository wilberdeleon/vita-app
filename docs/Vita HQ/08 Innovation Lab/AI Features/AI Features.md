# AI Features

**Category of:** [[Innovation Lab]]

**Scope:** AI-powered capabilities that aren't specifically Atlas's coaching persona — infrastructure-level intelligence woven into the product (smart scanning, auto-categorization, pattern detection, predictive suggestions). If the idea is "the coach does X," it belongs in [[Atlas Features]] instead; if it's "the app is smarter about Y" without a coaching voice attached, it belongs here.

**Examples of what fits:** food-photo recognition for logging, automatic meal-pattern detection, smart reminder timing, anomaly detection in weight/water logs, receipt/label OCR.

For the underlying AI strategy (model choice, cost, privacy, evaluation), see [[Future AI]] in the Engineering/AI documentation — that page covers *how* Vita builds AI; this category covers *which* AI features are proposed.
