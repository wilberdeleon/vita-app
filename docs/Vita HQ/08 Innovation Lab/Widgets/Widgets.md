# Widgets

**Category of:** [[Innovation Lab]]

**Scope:** OS-level surfaces outside the app itself — iOS/Android home-screen widgets, lock-screen widgets, Live Activities, complications. A distinct discipline from in-app UX: glanceable, low-interaction, platform-constrained.

**Examples of what fits:** a Today widget showing water/macro progress, a Journey Stage lock-screen widget, a Live Activity during an active fast or workout.

**Dependency note:** widgets need real, live data ([[Supabase & Database]]) to be worth building — mock data has no glanceable value. Expect this category to activate after early data sprints land.
