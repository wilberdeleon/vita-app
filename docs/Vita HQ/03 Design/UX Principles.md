# UX Principles

**What is this?** The experience rules that hold across every screen, regardless of theme or feature.

**Why does it exist?** Visual systems change (the theme question proves it); these behavioral principles don't. They translate the [[Core Principles]] and [[Product Philosophy]] into design decisions.

---

## The emotional contract

Every time someone opens Vita they should feel **motivated, in control, and proud of their progress**. Every screen should **reduce stress, not create it**. The register is a calm, confident coach — never a nagging tracker.

## The principles

1. **No guilt mechanics — ever.** No shame streaks, no red warnings for missed days, no punishment framing. Progress over perfection is a product law, not a tone suggestion.
2. **Friction is the enemy of consistency.** The core loop (log a meal, log water, check progress) must be achievable in seconds. Every added tap must justify itself against the VITA Promise.
3. **Transformation is always visible.** Screens answer "how am I changing?" — not just "what did I do?" ([[My Journey]], [[Journey Stages]]).
4. **Simplicity wins.** A feature needing a long explanation needs a redesign. New users should understand every primary screen immediately.
5. **Calm premium, not flashy.** No trend-chasing, no gamification aesthetics, no clutter. Quality bar: Apple / Oura / WHOOP ([[Design Bible]]).
6. **Consistency across the app.** Same tokens, same primitives, same motion vocabulary everywhere ([[Component Library]]). Domain colors carry meaning ([[Color System]]).
7. **Accessibility is quality.** Roles/labels on interactive elements (the dock already does this), contrast discipline, reduced-motion respect. "Quality is a feature" includes users with different abilities.
8. **Trust by design.** Health data is personal. Ask only for what's needed, explain why, and make control obvious ([[Settings]]).

## How to use this page

When reviewing a slice ([[Codex Review Checklist]]), these principles are the UX portion of the review. When a design debate stalls, check which principle it touches — most debates are already settled here.
